# 🍔 Food Delivery Data Analysis

## 📖 Overview

Food delivery platforms generate large amounts of customer and restaurant data every day. This project performs Exploratory Data Analysis (EDA) to discover valuable business insights using Python and Power BI.

---

## 🎯 Objectives

- Analyze customer ordering behavior
- Identify top-performing restaurants
- Discover popular cuisines
- Analyze revenue trends
- Study membership impact on sales
- Visualize business metrics

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel

---

## 📊 Dataset Features

- Order ID
- User ID
- Restaurant ID
- Order Date
- Total Amount
- Customer Name
- City
- Membership
- Restaurant Name
- Cuisine
- Rating

## 📊 Business Insights

- Hyderabad generated the highest revenue.
- Premium members placed more repeat orders.
- Biryani was the most ordered cuisine.
- Restaurants with higher ratings received more orders.
- Weekend sales were significantly higher than weekdays.
